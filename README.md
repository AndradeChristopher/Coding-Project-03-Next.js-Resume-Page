# Next.js Resume

## Description

This project is a responsive resume website built using Next.js (App Router) and Tailwind CSS.

## Technologies Used

- Next.js
- React
- Tailwind CSS
- TypeScript

## Installation
Note: You must first create your own generic Next.js application, then copy and paste the contents of this folder into the directory.
1. cd nextjs-resume
2. npm run dev
